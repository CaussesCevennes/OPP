A propos

L'observatoire photographique du paysage est un outil de gestion dont le principe consiste à photographier régulièrement le même point de vue. Les séries photographiques ainsi obtenues nous aident à mieux percevoir les évolutions paysagères.

Ce site utilise l'application VOPP développée par l'Entente Interdépartementale des Causses et des Cévennes. Le code source est ouvert et peut être réutilisé pour valoriser d'autres obversatoires photographiques.
